# La Cúpula Launchpad

¡Bienvenido a **La Cúpula Launchpad**! Este repositorio contiene el código fuente de nuestra plataforma. 

## Tecnologías Utilizadas

Este proyecto ha sido desarrollado utilizando tecnologías web modernas para garantizar el mejor rendimiento, escalabilidad y experiencia de usuario:

- **React 19**
- **TanStack Start & Router**
- **Vite**
- **Tailwind CSS 4**
- **TypeScript**

## Requisitos Previos

Asegúrate de tener instalado en tu entorno de desarrollo local:
- Node.js (versión 20.x o superior recomendada)
- npm o bun

## Instalación y Ejecución Local

Sigue estos pasos para levantar el entorno de desarrollo:

1. Instala las dependencias del proyecto:
   ```bash
   npm install
   ```

2. Ejecuta el servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Abre tu navegador web e ingresa a `http://localhost:8080`.

## Scripts Disponibles

- `npm run dev`: Inicia el servidor de desarrollo en modo watch.
- `npm run build`: Compila la aplicación para producción.
- `npm run preview`: Inicia un servidor local para previsualizar la compilación de producción.
- `npm run lint`: Ejecuta ESLint para analizar el código en busca de problemas.
- `npm run format`: Ejecuta Prettier para formatear el código.
